Adam

参数

(self, lr=0.001, beta_1=0.9, beta_2=0.999,epsilon=None, decay=0., amsgrad=False, **kwargs)

  • lr: float >= 0. 学习率。——0.001
  • beta_1: float, 0 < beta < 1. 通常接近于 1。——0.9
  • beta_2: float, 0 < beta < 1. 通常接近于 1。——0.999
  • epsilon: float >= 0. 模糊因子. 若为 None, 默认为 K.epsilon()。——None
  • decay: float >= 0. 每次参数更新后学习率衰减值。——0
  • amsgrad: boolean. 是否应用此算法的 AMSGrad 变种，来自论文 “On the Convergence of Adam and Beyond”。——false

公式

其中，$p_t = p_{t-1} - \frac{lr_t}{v_t^{1/2}+\epsilon}$ 是RMSprop过程，$m_t$ 相当于momentum

总结

算法 参数 公式 说明
SGD momentum = 0 $p_t = p_{t-1}- lr \times g_t$
SGD momentum>0 and nesterov=false $p_t = p_{t-1} + v_t = p_{t-1} \color{red}{-mlr\times\sum_{k=1}^{t-1} m^{t-k}g_k}-lr \times g_t$
SGD momentum>0 & nesterov = true $p_t=p_{t-1} + m \cdot v_{t} - lr \times g_t = p_{t-1}\color{red}{ - mlr\sum_{k=1}^tm^{t-k}g_k}-lr \times g_t$
Adagrad $p_t = p_{t-1} - \frac{lr \times g_t}{\color{red}{\sqrt{ \sum_{k=1}^{t}g_k^2} + \epsilon}}$
RMSprop $p_t =p_{t-1} - \frac{lr \times g_t}{a_t^{1/2} + \epsilon} $
$= p_{t-1}- \frac{lr \times g_t}{\color{red}{\sqrt{(1-rho)\sum_{k=1}^trho^{t-k}g_k^2} + \epsilon}}$		 基于Adagrad增加参数rho做了衰减
Adadelta $p_t=p_{t-1}-lr \times g_t {\color{red}{\times \frac {(da_t + \epsilon)^{1/2}} {(a_t + \epsilon)^{1/2}}}}$ 基于RMSprop 多了 $\sqrt{da_t+\epsilon}$
Adam $p_t = p_{t-1} -\color{red}{m_t} \times \frac{lr}{\color{blue}{v_t^{1/2}+\epsilon}}$ SGD+momentum 和RMSprop的结合
