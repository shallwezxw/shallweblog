SGD

Stochastic gradient descent optimizer：随机梯度下降优化器

参数

  • lr: 0.01
  • momentum: 0
  • decay: 0
  • nesterov: false

1) momentum = 0
$$
\begin{align*}
v_t= -lr\times g_t \
p_t = p_{t-1}- lr \times g_t \

\end{align*}
$$

2) momentum>0 and nesterov=false
$$
v_t = m.v_{t-1}-lr \times g_t
$$

$$
v_{t-1} = mv_{t-2} - lr \times g \
v_t = -\sum_{k=1}^{t}lr\times m^{t-k}g_k
$$

$$
p_t = p_{t-1} + v_t \
p_t = p_{t-1} + m \cdot v_{t-1} - lr\times g \
p_t = p_{t-1} \color{red}{-mlr\times\sum_{k=1}^{t-1} m^{t-k}g_k}-lr \times g_t
$$

3) momentum>0 & nesterov = true
$$
v_{t} = m\cdot v_{t-1}-lr \times g_t
$$

$$
p_t = p_{t-1} + m \cdot v_{t} - lr \cdot g_t \

= p_{t-1}\color{red}{ - mlr\sum_{k=1}^tm^{t-k}g_k}-lr \times g_t
$$

代码

Adagrad

参考

  • Adaptive Subgradient Methods for Online Learning and Stochastic Optimization

参数

  • lr: float >= 0. 学习率.
  • epsilon: float >= 0. 若为 None, 默认为 K.epsilon().
  • decay: float >= 0. 每次参数更新后学习率衰减值.

代码

公式
$$
a_{t} = a_{t-1} + g_t^2 \
a_{t} = \sum_{k=1}^{t}g_k^2 \
p_{t} = p_{t-1} - \frac{lr \times g_t}{a_t^{1/2} + \epsilon} \
= p_{t-1} - \frac{lr \times g_t}{\color{red}{\sqrt{ \sum_{k=1}^{t}g_k^2} + \epsilon}}
$$
有些地方该公式改为：
$$
p_{t} = p_{t-1} - \frac{lr \times g_t}{(a_t+ \epsilon)^{1/2} } \
= p_{t-1} - \frac{lr \times g_t}{\color{red}{(\sum_{k=1}^{t}g_k^2+ \epsilon)^{1/2} }}
$$

RMSprop

参考

  • rmsprop: Divide the gradient by a running average of its recent magnitude

参数

  • lr: float >= 0. 学习率。——0.001
  • rho: float >= 0. RMSProp梯度平方的移动均值的衰减率.——0.9
  • epsilon: float >= 0. 模糊因子. 若为 None, 默认为 K.epsilon()。——None
  • decay: float >= 0. 每次参数更新后学习率衰减值。——0

代码

公式
$$
a_{t} = rho \times a_{t-1} + (1 - rho) \times g_t^2 \
= (1-rho)\sum_{k=1}^trho^{t-k}g_k^2\
p_{t+1} = p_t - \frac{lr \times g_t}{a_t^{1/2} + \epsilon} \
= p_t - \frac{lr \times g_t}{\color{red}{\sqrt{(1-rho)\sum_{k=1}^trho^{t-k}g_k^2} + \epsilon}}
$$
有些地方该公式改为：
$$
p_{t} = p_{t-1} - \frac{lr \times g_t}{(a_t+ \epsilon)^{1/2} } \
= p_t - \frac{lr \times g_t}{\color{red}{((1-rho)\sum_{k=1}^trho^{t-k}g_k^2+ \epsilon)^{1/2} }}
$$

Adadelta

参考

  • Adadelta - an adaptive learning rate method

参数

  • lr: float >= 0. 学习率，建议保留默认值。
  • rho: float >= 0. Adadelta梯度平方移动均值的衰减率。
  • epsilon: float >= 0. 模糊因子. 若为 None, 默认为 K.epsilon()。
  • decay: float >= 0. 每次参数更新后学习率衰减值。

代码

公式
$$
a_{t} = rho \times a_{t-1} + (1 - rho) \times g_t^2 \
update = g_t \times \frac {(d_a_t + \epsilon)^{1/2}} {(a_t + \epsilon)^{1/2}} \
p_{t} = p_{t-1} - lr \times update = p_{t-1} - lr \times g_t {\color{red}{\times \frac {(d_a_t + \epsilon)^{1/2}} {(a_t + \epsilon)^{1/2}}} }\
d_a_{t+1} = rho \times d_a_t + (1-rho) \times update^2
$$

Adam

参考

  • [Adam - A Method for Stochastic Optimization](
    https://arxiv.org/abs/1412.6980v8)
    • On the Convergence of Adam and Beyond

参数

(self, lr=0.001, beta_1=0.9, beta_2=0.999,

*epsilon*=None, *decay*=0., *amsgrad*=False, ***kwargs*)
  • lr: float >= 0. 学习率。——0.001
  • beta_1: float, 0 < beta < 1. 通常接近于 1。——0.9
  • beta_2: float, 0 < beta < 1. 通常接近于 1。——0.999
  • epsilon: float >= 0. 模糊因子. 若为 None, 默认为 K.epsilon()。——None
  • decay: float >= 0. 每次参数更新后学习率衰减值。——0
  • amsgrad: boolean. 是否应用此算法的 AMSGrad 变种，来自论文 “On the Convergence of Adam and Beyond”。——false

代码

公式
$$
m_t = \beta_1 \times m_{t-1} + (1 - \beta_1) \times g_t \
v_t = \beta_2 \times v_{t-1} + (1 - \beta_2) \times g_t^2 \

p_t = p_{t-1} -m_t \times \frac{lr}{v_t^{1/2}+\epsilon}
$$
其中，$p_t = p_{t-1} - \frac{lr_t}{v_t^{1/2}+\epsilon}$ 是RMSprop过程，$m_t$ 相当于momentum

总结

算法 参数 公式 说明
SGD momentum = 0 $p_t = p_{t-1}- lr \times g_t$
SGD momentum>0 and nesterov=false $p_t = p_{t-1} + v_t = p_{t-1} \color{red}{-mlr\times\sum_{k=1}^{t-1} m^{t-k}g_k}-lr \times g_t$
SGD momentum>0 & nesterov = true $p_t=p_{t-1} + m \cdot v_{t} - lr \times g_t = p_{t-1}\color{red}{ - mlr\sum_{k=1}^tm^{t-k}g_k}-lr \times g_t$
Adagrad $p_t = p_{t-1} - \frac{lr \times g_t}{\color{red}{\sqrt{ \sum_{k=1}^{t}g_k^2} + \epsilon}}$
RMSprop $p_t =p_{t-1} - \frac{lr \times g_t}{a_t^{1/2} + \epsilon} = p_{t-1}- \frac{lr \times g_t}{\color{red}{\sqrt{(1-rho)\sum_{k=1}^trho^{t-k}g_k^2} + \epsilon}}$ 基于Adagrad增加参数rho做了衰减
Adadelta $p_t = p_{t-1} - lr \times g_t {\color{red}{\times \frac {(d_a_t + \epsilon)^{1/2}} {(a_t + \epsilon)^{1/2}}} }$ 基于RMSprop 多了 $\sqrt{d_a_t+\epsilon}$
Adam $p_t = p_{t-1} -\color{red}{m_t} \times \frac{lr}{\color{blue}{v_t^{1/2}+\epsilon}}$ SGD+momentum 和RMSprop的结合
